# 評価ハーネス — 指標定義

> [research-overview.md](research-overview.md) §5（検証設計）の具体化。全手法ファミリ（A〜E）を**共通の土俵**で比較するための、出力正規化・座標系規約・指標・集計の定義。M2 で実装する。
>
> - ステータス: Draft v0.1
> - 最終更新: 2026-06-23
> - 評価対象: 正準座標系における全周囲統合点群 $\hat{S}$（最終出力）と、副産物の個体姿勢 $\{\hat{T}_i\}$

---

## 1. 設計原則

1. **出力の正規化**: どのファミリ（点群直出力 / SDF / 放射輝度場 / 生成）でも、最終的に**点群 $\hat{S}$ に統一**してから評価する（陰関数・体積場は標準密度で表面サンプリング）。→ 全手法が同一指標で比較可能。
2. **ゲージ自由度の明示**: 復元される正準座標系は**手法が任意に選ぶ**ため、GT と直接は比較できない。**グローバル整列（§3）後**に幾何指標を測る。
3. **誤差の3層分解**: 「被覆限界（A2）／レジストレーション誤差／融合・表現誤差」を分離して測る（§6）。どこがボトルネックかを切り分けられる設計にする。
4. **対称性の局所化**: 再構成の集合指標（CD/F-score 等）は**対称性に不変**（点集合間の最近傍距離なので）。対称群の対応が要るのは**姿勢指標だけ**（§7）。
5. **スケール既知**: 入力は計量深度センサ由来＝**絶対スケール既知**。整列は既定で剛体 $SE(3)$（スケール固定）。スケールフリー手法のみ $Sim(3)$ をオプション許可（§3）。

---

## 2. 前処理と表記

### 記号
- $S_{gt}$: GT メッシュからサンプリングした正準 GT 点群。
- $\hat{S}$: 手法が復元した全周囲点群（評価対象）。
- $\{P_i\}$: 各インスタンスの観測部分点群、$\{T_i\}$: GT 個体姿勢（世界→正準）、$\{\hat{T}_i\}$: 推定姿勢。
- $L$: 物体スケール = $S_{gt}$ のバウンディングボックス対角長（指標の正規化基準）。

### 前処理（公平性のため全手法・GT に同一適用）
- **GT 点群生成**: GT メッシュを **Poisson-disk サンプリング**で一様に取得（既定 $|S_{gt}| = 50\text{k}$ 点）。法線も保持。
- **密度正規化**: $\hat{S}$ と $S_{gt}$ を**共通ボクセルサイズ $v$** でダウンサンプル（既定 $v = 0.5\%\,L$）してから指標計算。→ 点を大量に吐く手法が有利/不利にならないようにする。
- **外れ値**: 評価前に統計的外れ値除去は**かけない**（精度指標 Accuracy が外れ点を拾うのは正しい挙動）。前処理での除去は手法側の責務とする。

---

## 3. グローバル整列プロトコル（ゲージ自由度の吸収）★中核

$\hat{S}$ を $S_{gt}$ に重ねる剛体変換 $G \in SE(3)$ を求めてから全指標を測る。任意座標系の差を吸収しつつ、**対称物体での多義性**にロバストにする：

```
1. 特徴ベース粗整列: FPFH + RANSAC で G の初期候補を複数生成（Open3D）
2. 多重スタート: 初期候補ごと + 物体対称群の各元から ICP（point-to-plane）で精緻化
3. 採用基準: 整列後の Chamfer Distance を最小化する G* を採用
```

- **対称物体**: 複数の妥当な整列が存在するが、CD は全て同値になるため、min-CD 採用で自然に処理される（再構成指標は対称性に不変、原則4）。
- **スケール**: 既定で $s=1$ 固定。スケールフリー手法のみ Umeyama で $s$ も推定する $Sim(3)$ 整列を**明示フラグ付き**で許可し、報告時に区別する。
- **報告**: 幾何指標はすべて「$G^*$ 整列後」の値。整列前の値は座標系が任意なので報告しない。
- **注意（楽観性）**: min-CD で整列してから CD を報告するのは標準手法だが楽観方向に働く。全手法・oracle に**同一手順**を適用して相対比較の公平性を担保する。

## 4. 再構成品質指標（L1: $\hat{S}$ vs $S_{gt}$）

最近傍距離 $d(x, Y) = \min_{y \in Y}\lVert x - y\rVert_2$ を用いる。

| 指標 | 定義 | 意味 | 主/副 |
|------|------|------|:---:|
| **Accuracy**（精度方向） | $\frac{1}{\lvert\hat S\rvert}\sum_{x\in\hat S} d(x, S_{gt})$ | 偽の点・ノイズ点が表面からどれだけ外れるか | 主 |
| **Completeness**（再現方向） | $\frac{1}{\lvert S_{gt}\rvert}\sum_{y\in S_{gt}} d(y, \hat S)$ | GT 表面のうち復元で埋まっていない量 | 主 |
| **Chamfer Distance (CD)** | $\text{Accuracy} + \text{Completeness}$（または平均） | 双方向の総合一致度。**主指標** | 主 |
| **Precision@τ** | $\frac{1}{\lvert\hat S\rvert}\lvert\{x\in\hat S : d(x,S_{gt})<\tau\}\rvert$ | τ 以内に乗る復元点の割合 | 主 |
| **Recall@τ** | $\frac{1}{\lvert S_{gt}\rvert}\lvert\{y\in S_{gt} : d(y,\hat S)<\tau\}\rvert$ | τ 以内で覆われた GT 表面割合 | 主 |
| **F-score@τ** | $\dfrac{2\,P_\tau R_\tau}{P_\tau + R_\tau}$ | 精度と被覆の両立。**主指標** | 主 |
| **Normal Consistency** | $\frac{1}{\lvert\hat S\rvert}\sum_x \lvert\langle n(x), n(\text{NN}_{S_{gt}}(x))\rangle\rvert$（双方向平均） | 表面向きの一致（法線がある時） | 副 |

- **τ の既定**: $\{0.5\%,\,1\%,\,2\%\}\times L$ の3点 + センサ仕様が分かれば絶対値（mm）も併記。F-score は **τ に対する曲線**で報告（AUC も可）。
- **CD の変種**: 主は L1（最近傍ユークリッド距離の平均）。二乗版 CD-L2 も補助的に併記可（外れ点に敏感）。
- 距離は **絶対値（mm）と相対値（%$L$）の両方**で報告し、物体間比較を可能にする。

## 5. レジストレーション品質指標（L2: $\{\hat T_i\}$ vs $\{T_i\}$）

姿勢は再構成のための中間量だが、診断として測る。ゲージ自由度のため、まず**単一のグローバル整列 $G^*$**（§3）を推定姿勢系へ適用 $\tilde T_i = G^* \!\circ \hat T_i$ してから残差を測る。

- **回転誤差（対称性考慮）**: $e_R^{(i)} = \min_{R_s \in \mathcal{G}_{sym}} \angle\!\big(\tilde R_i,\; R_i R_s\big)$、ただし $\angle(A,B)=\arccos\frac{\mathrm{tr}(AB^\top)-1}{2}$。
- **並進誤差**: $e_t^{(i)} = \lVert \tilde t_i - t_i\rVert / L$（$L$ 正規化、%表示）。
- **レジストレーション成功率**: $e_R<\theta$ かつ $e_t<d$ を満たす個体割合。既定しきい値 $(\theta,d)=(5^\circ,\,2\%L)$、補助で $(10^\circ,5\%L)$。
- **報告**: 回転・並進の mean/median/90%ile、成功率、ヒストグラム。対称群 $\mathcal{G}_{sym}$ は BOP `models_info.json` の `symmetries_discrete/continuous` を GT として利用（合成では既知）。

---

## 6. 被覆性 A2 と誤差3層分解（L3）★研究貢献に直結

「全周囲が観測されているか（A2）」を**手法から切り離して**測り、再構成誤差の内訳を分解する。

### 6.1 観測被覆（oracle / 上界）
GT 姿勢で観測点を融合した和集合 $U = \bigcup_i T_i^{-1}(P_i)$ を作り、$\hat S$ を $U$ に置き換えて Recall@τ を計算 = **観測被覆率**。これは「完璧に位置合わせできた時の到達上限」で、**データ（物体形状・$N$・接地姿勢分布）だけで決まる**。
- これ自体が A2 の成立度を定量化する研究知見。

### 6.2 誤差の3層分解
| 層 | 測り方 | 切り分けるもの |
|----|--------|----------------|
| ① 被覆限界 | $1 - \text{Recall}^{oracle}_\tau$ | A2 が破れて**原理的に**埋まらない領域 |
| ② レジストレーション誤差 | （oracle 融合）−（推定姿勢 $\hat T_i$ で融合）の差 | 位置合わせのズレに起因する劣化 |
| ③ 融合・表現誤差 | （推定姿勢融合）−（最終 $\hat S$）の差 | stage④⑤（融合・補完・表現）の劣化 |

→ ある手法のスコアが低い時、原因が「観測不足」「位置合わせ」「表現」のどこかを一目で切り分けられる。

### 6.3 被覆 vs インスタンス数 $N$ 曲線
$N$ を変えて観測被覆率（6.1）と最終 F-score を測り、**「全周囲を埋めるのに必要な $N$」**を物体クラス別に提示。

### 6.4 未観測領域マップ
$S_{gt}$ 各点が観測被覆されたか否かを物体表面に可視化し、**接地姿勢の偏りで観測されにくい領域**を示す（A2 リスクの定性根拠）。

## 7. 対称性の扱い（まとめ）

- **再構成指標（§4）・被覆（§6）**: 点集合間の最近傍距離 → **対称性に不変**。特別扱い不要。
- **姿勢指標（§5）のみ**: 対称群 $\mathcal{G}_{sym}$ にわたる最小を取る（ADD-S 系の発想）。連続対称（回転体）は群を離散サンプリングして近似。
- 整列（§3）の多義性も min-CD で吸収されるため、対称性が再構成スコアを不当に下げることはない。

---

## 8. 集計・レポート

- **集計単位**: シーン × 物体。mean ± std / median / 物体別内訳。
- **アブレーション軸**（[research-overview.md](research-overview.md) §5.3）別に層別集計: $N$ / 対称性クラス / 遮蔽率 / ノイズ強度 / 接地姿勢偏り。
- **必ず oracle を併記**: 各手法のスコアの隣に 6.1 の上界を置き、「伸びしろ」と「原理限界」を可視化。
- **図**: F-score–τ 曲線、CD 棒グラフ、被覆–$N$ 曲線、未観測領域マップ。
- **シード固定**で再現可能な数表を CSV/JSON で出力（スペック駆動・回帰比較用）。

## 9. 実装メモ（既定値つき）

| 項目 | 既定 | ライブラリ |
|------|------|-----------|
| GT 点群サンプリング | Poisson-disk, 50k 点, 法線付き | trimesh / Open3D |
| 密度正規化ボクセル $v$ | $0.5\%\,L$ | Open3D |
| 最近傍探索 | KD-tree | Open3D / scipy.cKDTree |
| 粗整列 | FPFH + RANSAC（多重スタート） | Open3D |
| 精整列 | point-to-plane ICP | Open3D |
| τ | $\{0.5,1,2\}\%\,L$（+ mm） | — |
| 成功率しきい値 | $(5^\circ, 2\%L)$ / $(10^\circ, 5\%L)$ | — |
| CD/F-score の GPU 実装 | 大規模時 | PyTorch3D |
| 対称群 | BOP `models_info.json` | BOP toolkit |

**インタフェース案**: `evaluate(S_hat, S_gt, poses_hat=None, poses_gt=None, sym_group=None, partials=None) -> dict` が、整列 → L1 → （姿勢あれば）L2 → （観測ありなら）L3 を一括計算し、数表を返す。oracle 融合（6.1）は `partials + poses_gt` から内部生成。
