## 0. 一行サマリ

同一種類の剛体部品がバラ積みされた段ボールを**1ショット計測**して得られる**2.5D 片面部分点群の集団**を、**個体間レジストレーション**によって共通正準座標系へ統合し、1物体の**全周囲モデル（最終表現は点群）**を復元する。

---

## 1. 背景と動機

- FA（Factory Automation）現場では、**1種類の物体が大量に段ボールへバラ積み**されたシーンが頻出する。
- 上方からの距離計測（深度センサ等）で点群を取得すると、各部品は遮蔽・視点制約により**2.5D（片面）部分点群**としてしか観測されない。
- 一方で、バラ積みは**同一形状の剛体が多数・多様な姿勢**で存在する。個々の見えは部分的でも、**集団全体では全周囲のあらゆる面が何らかの姿勢で観測されている**可能性が高い。
- この「**別個体だが幾何的に同一**」という性質を、1物体のマルチビュー観測の代用として使えば、**追加撮影なし（ワンショット）で全周囲モデルを復元**できるのではないか、というのが本研究の着想。

**実務的価値**: 復元した全周囲モデルは、把持計画・6Dof姿勢推定器の学習データ生成・外観検査・デジタルツイン等の下流タスクに再利用できる。CAD が無い／入手困難な部品でも、現場計測だけでモデルを起こせる点が狙い。

---

## 2. 問題定義（フォーマル）

### 入力
- 段ボール内に未知の剛体 $S$ の同一インスタンスが $N$ 個バラ積みされたシーンを、上方から1回計測した 2.5D 点群 $P$。
- 個体分離後、$N$ 個の部分点群 $\{P_i\}_{i=1}^{N}$。各 $P_i$ はインスタンス $i$ の**可視（主に上面側）表面**を、未知の姿勢 $T_i \in SE(3)$ で観測したもの。

### 出力
- **正準座標系における全周囲の統合点群** $\hat{S}$（$\partial S$ の完全表面を近似）。
- 副産物として個体姿勢 $\{\hat{T}_i\}$。

### コア定式化（鶏卵問題）
共通の表面に全部分ビューが整合するように、**姿勢と形状を同時推定**する：

$$
\min_{\{T_i\},\, S} \; \sum_{i=1}^{N} d\!\left(T_i^{-1} P_i,\; S\right)
$$

形状 $S$ を作るには位置合わせ $\{T_i\}$ が要り、位置合わせには $S$（参照モデル）が要る → **群（groupwise）レジストレーション＋形状推定の同時最適化**が核心。

### 前提（Assumptions）
| ID | 前提 | 備考・リスク |
|----|------|------|
| A1 | 同一剛体: 全インスタンスは同一形状 $S$ | 製造公差・個体差は無視できる程度と仮定（要検証） |
| A2 | 被覆性: $\bigcup_i T_i^{-1}(\text{visible}_i) \approx \partial S$ | **安定接地姿勢の偏りで破れ得る**（§5.2）。本研究の最重要前提 |
| A3 | 未知物体: CAD/テンプレート無し | 既知 CAD があれば問題は大幅に簡単化（比較用ベースラインに使える） |
| A4 | 分離可能: 個体セグメンテーションが得られる | サブ課題として扱う（§4 stage①、スコープ判断は §6） |
| A5 | 剛体・非変形 | 軟性・可動部品は対象外 |

## 3. 新規性と位置づけ

本研究は既存の複数問題系の交差点にあり、その**組み替え**に新規性がある。

| 既存問題系 | 本研究との違い |
|-----------|----------------|
| マルチビュー3D復元 / SfM | 「同一物体の複数視点」ではなく「**別個体だが同一形状**」を視点の代用にする |
| グローバル点群レジストレーション | 参照モデル無しで $N$ 個を**同時**に正準系へ揃える（pairwise でなく groupwise） |
| 形状補完 (Shape Completion) | 単一部分入力の補完ではなく、**多数の部分観測の融合**で完全化 |
| カテゴリレベル正準化 | カテゴリ事前ではなく、**眼前のインスタンス群そのもの**から正準形状を立ち上げる |
| Bin-picking の姿勢推定 | 姿勢推定が目的ではなく、姿勢は**全周囲復元のための中間量** |

**一言で**: 「1ショットのバラ積み観測を、未知物体の groupwise multi-view と見なして全周囲復元する」。

### 新規性確認（M1 文献調査、詳細は [literature-survey.md](literature-survey.md)）
- **本問題設定をそのまま提案した先行研究は確認されず**、新規性ギャップは保たれている。
- 中核アイデア（繰り返し同一インスタンスを補完的観測として活用し joint reconstruction）の**最近接は FurnSet（arXiv:2604.20093, 2026）**。ただし「単一RGB・屋内家具・カテゴリ学習済み生成事前分布・GT同一性ラベル依存」であり、本研究の「**深度/点群・産業バラ積み・参照モデル無し/CAD未知・groupwise registration**」とはモダリティ/ドメイン/参照依存性/手法の4軸で異なる（=新規性の主張点）。
- 注意: 新規性は「アイデア未踏」ではなく「**この組み合わせが未踏**」として主張する。近接プレプリント（FurnSet 2026-04 / Lookalike3D 2026-03）は査読状況を継続監視。

## 4. 共通パイプラインと手法ファミリ（比較探索方針）

手法は1本化せず、**共通6ステージのパイプライン**を定義し、差別化の本質である **stage③（個体間正準位置合わせ）** と **stage⑤（表現・補完）** を差し替えて比較する。これにより「最終出力＝点群」「中間表現は手法依存」を統一土俵で評価できる。

### 共通パイプライン
```
① 個体セグメンテーション      段ボール点群 → {インスタンス領域}
② 部分点群抽出・前処理        {P_i}（法線推定・ダウンサンプル・外れ値除去）
③ 個体間正準位置合わせ        {P_i} → {T_i}（共通正準座標系へ）   ★差別化点
④ 融合・集約                  ∪ T_i^{-1} P_i → 統合点群（重複統合・密度均一化）
⑤ 表現・補完                  穴埋め・平滑化（中間表現は手法依存）  ★差別化点
⑥ 最終全周囲点群 \hat{S}       正準点群を出力（+ 評価）
```

### 手法ファミリ候補（stage③/⑤ の実装が異なる）
| ID | ファミリ | stage③（位置合わせ） | stage⑤（表現/補完） | 中間表現 | 長所 / 短所 |
|----|---------|---------------------|----------------------|----------|-------------|
| A | 古典幾何 | 特徴量ベース global registration + groupwise ICP 精緻化 | 密度融合 + Poisson/球面投票等 | 点群/メッシュ | 解釈可能・最短でベースライン化 / 対称性・初期値に弱い |
| B | 学習正準化 | 正準姿勢回帰（インスタンス→正準系） | 統合点群を直接出力 | 点特徴 (PointNet系) | 対称性を学習で吸収 / 学習データと汎化が課題 |
| C | 陰関数場 (DeepSDF系) | 潜在形状 $z$ と $\{T_i\}$ の同時最適化 | SDF → 等値面サンプリングで点群化 | SDF | 欠損補完が自然 / 最適化が重く局所解 |
| D | 微分可能レンダリング (NeRF系) | 各個体姿勢を可微分に最適化 | 体積場 → 表面サンプリング | 放射輝度/密度場 | 視点整合が強い / 深度のみ入力での定式化に工夫要 |
| E | 生成事前分布 | A〜Dいずれかで初期化 | 拡散/生成モデルで補完・洗練 | 生成モデル潜在 | 未観測面の補完力 / 事前学習依存・幻影リスク |

> 比較の公平性のため、**stage①②④⑥と評価指標は全ファミリ共通**とし、③⑤のみ差し替える設計にする。

### 各ファミリの具体手法（M1 で特定、詳細・出典は [literature-survey.md](literature-survey.md) §5）
| ファミリ | 比較/土台となる具体手法 | 留意 |
|----------|------------------------|------|
| A 古典幾何 | ペアワイズ: TEASER++ / KISS-Matcher(FPFH) ＋ 多視点: **SGHR (pose-graph+IRLS)** | baseline 中核（[baseline-classical-geometry.md](baseline-classical-geometry.md)） |
| B 学習正準化 | ConDor / SE(3)-equivariant self-supervised | カテゴリレベル制約（未知カテゴリ・対称で弱い）が差別化の論点 |
| C 陰関数場 | DeepSDF / MV-DeepSDF | カテゴリ事前依存。参照無し化が工夫所。MV-DeepSDFの「経時観測」とは構造が違う点に注意 |
| D 微分可能/joint最適化 | correspondence-free depth-guided joint opt / ODIN | 深度ベース joint 最適化。NeRF-深度特化は補足調査(M1.5) |
| E 生成事前分布 | PVD（拡散補完） | 参照モデル不要で親和的 |

> **推奨初期比較**: A（TEASER++/KISS-Matcher + SGHR系多視点最適化）を baseline、最初の相手に **SGHR/ODIN** と **PVD** を据える。

## 5. 検証設計

### 5.1 評価指標（最終点群 $\hat{S}$ に対して）
- **Chamfer Distance (CD)**: $\hat{S}$ と GT 点群間の双方向最近傍距離（主指標）。
- **F-score@τ**: 許容距離 τ における precision/recall の調和平均（被覆と精度の両立を見る）。
- **Accuracy / Completeness**: 片方向距離の分離評価（精度 vs 抜け漏れ）。
- **Coverage ratio**: GT 表面のうち統合点群でカバーされた面積比（**A2 前提の検証**に直結）。
- **姿勢誤差**: $\{\hat{T}_i\}$ の回転・並進誤差（**対称性を考慮した剰余**で評価）。

### 5.2 前提 A2（被覆性）の明示的検証
- 安定接地姿勢の分布を計測・シミュレートし、**どの面が観測されにくいか**を定量化。
- 「全周囲網羅」が成立する条件（必要インスタンス数、物体形状クラス）をアブレーションで明らかにする。これ自体が研究の知見になる。

### 5.3 アブレーション軸
インスタンス数 $N$ / 対称性クラス / 遮蔽率 / センサノイズ・欠損 / 接地姿勢の偏り。

## 6. データ戦略（合成先行）

- **フェーズ1: 合成データ先行**
  - 物理シミュ（剛体落下）で段ボールへのバラ積みシーンを生成 → 上方から深度レンダリング → 2.5D 点群化。
  - **GT メッシュ・GT 個体姿勢・GT セグメンテーション**が同時に得られるため、全指標を定量評価可能。
  - 物体ソース候補: 手続き生成形状 / 公開 CAD パーツ集 / FA 模擬部品。対称性・凹凸の難易度を段階化。
  - センサ実機を模した**ノイズ・欠損モデル**を後段で付与し、現実とのギャップを制御。
- **フェーズ2（後続）: 実機計測**
  - 実深度センサで段ボールを撮影し、ドメインギャップ検証。GT は別手段（高精度スキャン等）で取得。

> 公開データセットの流用可否（バラ積み + GT メッシュ付き）は別途調査（§8 未決）。

## 10. 用語

- **2.5D / 片面点群**: 単一視点の深度から得られる、物体の見えている面だけの点群。
- **個体間レジストレーション (groupwise registration)**: 参照モデル無しに複数の部分観測を共通座標系へ同時整合させる処理。
- **正準座標系 (canonical frame)**: 復元対象1物体に固定した基準座標系。全インスタンスをここへ揃える。
- **被覆性 (coverage)**: 全インスタンスの可視面の和集合が物体の全表面を覆う度合い。
